## Install

Overall install information.

### Install Node.js

<https://nodejs.org/>


#### Mac & Homebrew
```
brew install node
```

### Install Node Modules

```bash
$ npm install electron
$ npm install electron-packager
$ npm run package-mac
```

### Configure Language

```
export LANG=en_US.UTF-8
export LANG=en_GB.UTF-8
```

### Build
```
$ npm run package-<mac|windows|linux>
```
